### 🧠 FlowOne — AI Agent Orchestration Platform
**Role:** Lead Architect & Developer &nbsp;·&nbsp; **Timeline:** Oct 2024 – Present &nbsp;·&nbsp; [Live →](https://useflowone.com)

AI-native SaaS platform enabling businesses to build, deploy, and manage autonomous AI agents — unifying Agent Studio, Workflow Studio, and Voice AI into a single production system.

**What I built**
- Real-time WebSocket voice pipeline (STT → LLM → TTS) via ElevenLabs + Twilio
- Omnichannel automation across WhatsApp Business API, Shared Inbox, and Web Chat
- Multi-tenant architecture with isolated workspaces for enterprise clients
- Secure authentication and modular backend services for scalable deployment

### 🕸️ PinMind — Multi-Agent Orchestration Framework
**Role:** Lead Developer &nbsp;·&nbsp; **Timeline:** Oct 2024 – Present

A reusable framework for building context-aware, autonomous agent systems — designed to handle multi-step reasoning and long-running tasks without losing context.

**What I built**
- Context-aware agent routing, planning, and task orchestration on LangGraph
- Modular agent workflows with shared memory and extensible tool integration
- Memory-aware execution for context-rich, multi-step task handling

### 🔥 Fire Drone — Autonomous Wildfire Detection
**Role:** Solo Developer · Capstone Project

Autonomous wildfire detection system built to identify fire outbreaks from aerial drone footage in real time.

**What I built**
- Trained a YOLOv8 computer vision model on a combined Kaggle/Roboflow dataset — **~95% detection accuracy**
- FastAPI backend streaming real-time detection data over WebSockets
- React dashboard for live monitoring of detection results
